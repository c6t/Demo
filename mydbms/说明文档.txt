sql语句(所有语句以;结束)：

创建数据库：create database 数据库名
使用数据库：use database 数据库名
删除数据库：delete database 数据库名
列出所有数据库：show databases
列出数据库中的所有表名：show tables

创建表：create table 表名 (属性名1 数据类型，数据名2 数据类型,......)
删除表：delete table 表名
向表中插入数据：insert into 表名(属性名1，属性名2，...) values(值1，值2，...)
从表中删除数据：delete from 表名 where 属性x=值x
从表中查询数据：select * from 表名
                select 属性1，属性2,... from 表名
                select 属性1，属性2，...from 表名 where 属性x=值x
更新表中数据：update 表名 set 属性m=值m，属性n=值n where 属性x=值x

退出：quit
